# Auto Prime - projeto pronto para GitHub e Vercel

## Como testar no computador
1. Instale o Node.js
2. Abra a pasta no terminal
3. Rode:

```bash
npm install
npm run dev
```

## Como publicar
1. Suba esta pasta inteira para um repositório no GitHub
2. Na Vercel, clique em **Add New Project**
3. Escolha o repositório
4. Clique em **Deploy**

A Vercel deve detectar automaticamente que é um projeto Vite + React.
